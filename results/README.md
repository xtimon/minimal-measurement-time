# Директория результатов

Эта директория содержит все результаты симуляций, созданные пакетом `measurement-time-simulator`.

## Типы файлов

- `simulation_results_*.txt` - Результаты одиночных симуляций
- `system_comparison_*.txt` - Сравнение нескольких систем
- `large_scale_simulation_stats_*.txt` - Статистика крупномасштабных симуляций
- `final_simulation_report_*.txt` - Итоговые отчеты
- `comprehensive_final_report_*.txt` - Комплексные отчеты
- `executive_summary_*.txt` - Исполнительные резюме

## Примечание

Все файлы в этой директории игнорируются Git (см. `.gitignore`), за исключением этого файла README.md и файла `.gitkeep`, который гарантирует, что директория отслеживается в репозитории.

Вы можете безопасно удалять файлы результатов по мере необходимости.
